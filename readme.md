# Fundación NovaVida DBT1 - Plataforma Web

Este repositorio contiene el código fuente del sitio web oficial de la **Fundación NovaVida DBT1**. El proyecto tiene como objetivo proporcionar información, educación y herramientas de contacto para personas con Diabetes Tipo 1, fomentando una comunidad de apoyo.

## Descripción del Proyecto

La página web es un portal informativo y funcional que permite a los usuarios:

- Acceder a información sobre cuidado diario, nutrición y tecnologías de la diabetes.
- Interactuar con listas dinámicas mediante JavaScript/jQuery.
- Enviar consultas a través de un formulario de contacto con validación en tiempo real.
- Visualizar contenido multimedia y recursos educativos.

## Tecnologías Utilizadas

Este proyecto fue desarrollado utilizando las siguientes tecnologías y librerías:

- **HTML5:** Estructura semántica del sitio.
- **CSS:** Estilos personalizados (`styles.css` y `nih_styles.css`).
- **Bootstrap 5:** Framework para el diseño responsivo (Grillas, Cards, Modales, Navbar, Botones).
- **Bootstrap Icons:** Iconografía para mejorar la interfaz de usuario.
- **JavaScript:** Lógica del lado del cliente (Menú activo, funciones globales).
- **jQuery:** Librería utilizada para:
  - Manipulación del DOM (Agregar elementos a listas dinámicamente).
  - Efectos visuales (`slideToggle` para ocultar/mostrar secciones).
  - Validación de formularios (Expresiones regulares para emails y nombres).
  - Manejo de eventos (Clicks, Submit, Input).

## Funcionalidades Clave

1.  **Formulario Inteligente:** Validación de campos obligatorios, formato de correo y contador de caracteres para el mensaje, desarrollado con jQuery.
2.  **Navegación Responsiva:** Barra de navegación que colapsa en móviles y resalta la página activa automáticamente.
3.  **Interactividad:** Botones que modifican el contenido de la página sin recargarla (añadir elementos a listas, mostrar/ocultar formulario).
4.  **Componentes Bootstrap:** Uso de Modales para confirmaciones para organizar la información.

## Estructura de Archivos

- `index.html`: Página principal.
- `assets/css/`: Hojas de estilo.
- `assets/js/`: Scripts de lógica y librerías.
- `assets/img/`: Recursos gráficos.
- `index.html del 2 al 9`: Páginas secundarias que complementan la página principal, vinculadas a cada opción del menú.
